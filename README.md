## Generating documentation YAML

You can utilize the `generate_docs` shell script to generate the
merged documentation YAML file.

### Dependencies

- Node 8.0.2 (or later)
- NPM 6.0.0 (or later)

### Running the script

```bash
$ ./generate_docs
```

> The command will install `json-refs` package globally, if it does not exist.
